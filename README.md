 # 5-Star Rating System

 This is a code bundle for 5-Star Rating System. The original project is available at https://www.figma.com/design/VsNsNXy0o5ak9O1AKOO8q1/5-Star-Rating-System.

 ## Running the code

 Run `npm i` to install the dependencies.

 Run `npm run dev` to start the development server.

 ## Deploying to Vercel

 - Import this repository into Vercel.
 - When prompted, set the **Root Directory** to `5-Star Rating System`.
 - Install Command: `pnpm install` (or `npm install` if you prefer).
 - Build Command: `pnpm build` or `npm run build`.
 - Output Directory: `dist`.

 A `vercel.json` file is included to ensure the project is served as a static site and routes all requests to `index.html` for client-side routing.
